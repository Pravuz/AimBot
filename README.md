# AimBot
Undergrad Robotics project. Check out our youtube (www.youtube.com/channel/UCTwyZwKfavoUkVI_xjq9n3A) if you're interested.
This is a Bachelors project on constructing an automated camera-rig.<br /><br />


Folder contents:

Arduino -> Arduino mega 2560, the "brain" of the rig <br /><br />
Brugi -> Brushless DC-motor controller, based on Sine-wave pwm on 2-axis. <br /><br />
Common -> Files that all source code implements. protocols etc.<br /><br />
Desktop -> Windows configuration program <br /><br />
Pixy -> Camera module with a dual-core arm cpu for image processing. <br /><br />
Release - > Executable's etc for end-user. <br /><br />
